## 🧭 What It Is
Sierra Chart for Windows is a desktop platform that pairs a fast charting engine with detailed order flow analysis. It is built for traders who want to see how volume trades at each price level, using footprint (numbers bars) charts alongside standard candlestick and tick charts. The application is lightweight, reliable during long trading sessions, and highly configurable, so you can shape charts, studies, and workspaces around the way you read the market.

> Sierra Chart focuses on the data itself: fast charts, clear order flow, and the freedom to build custom studies rather than being locked into a fixed set of tools.

## ⭐ Key Features
- 📈 **Footprint order flow charts** — Study numbers bars that show traded volume at each price level for granular intrabar reading.
- ⚡ **Lightweight, fast performance** — Keep many charts and studies open with quick redraws and low overhead.
- 🧮 **Extensive indicator library** — Reach for a broad set of Sierra Chart indicators, then extend it with your own custom studies.
- 🎛️ **Chart-based order entry** — Submit and manage orders directly from charts with configurable trading controls.
- 🎨 **Flexible workspaces** — Save layouts, colors, and timeframes so your setup loads exactly how you like it.

## 💬 Questions
**Is Sierra Chart free?**
Sierra Chart is a paid subscription service with tiered packages, and the tier you choose determines which features and data you can access. A trial may be available for evaluation. It is charting and trading software, not a source of guaranteed returns.

**Which versions of Windows are supported?**
Sierra Chart is a native Windows desktop application intended for current 64-bit Windows versions. Using at least the OS vendor's minimum supported version keeps performance and compatibility strong.

**How does Sierra Chart compare to Bookmap?**
Bookmap is an order-flow-focused platform centered on a heatmap view of the order book, while Sierra Chart is a broader charting and trading platform that includes footprint charts, a large indicator set, and custom studies. Many traders evaluate both to see which order flow presentation fits their style.
